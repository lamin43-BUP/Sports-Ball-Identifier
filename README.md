# üèÄ Sports Ball Detection App
<br>
An Android application that automatically detects and identifies 12 different types of sports balls using a machine-learning model 
trained with Teachable Machine. The app supports real-time camera capture and gallery image selection, allowing users to classify
sports balls quickly and accurately through a simple and user-friendly interface.
<br>
<br>

## Features:
* Detects 12 sports ball types using a trained TensorFlow Lite model
* Supports real-time camera photo capture
* Allows detection from gallery images
* Shows confidence score for each prediction
* Simple and clean interface for easy use

<br>

## Prerequisites:
* Android Studio (latest version)
* Android SDK 21+
* Java Development Kit (JDK) 11+
* Android device/emulator with camera
* Install the required packages using pip


<br>

## Technical Specifications: 
* Programming Language: Java
* ML Framework: TensorFlow Lite
* Minimum SDK: Android 5.0 (API 21)
* Target SDK: Android 13 (API 33)
* Permissions: Camera and Gallery access
